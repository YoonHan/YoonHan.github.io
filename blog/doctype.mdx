> html 파일 맨 위에 선언되어있는 **`<!DOCTYPE html>`** 에 대해 알아본다.

## DOCTYPE 이란?

**Document Type** 의 약자다. 이 DOCTYPE 이라는 것은 항상 DTD(Document Type Definition)와 연관되어있다. <br />
DTD는 HTML 문서를 어떤 방식으로 구조화할 것인지를 결정한다. 예를 들면, button 태그 안에 span 태그는 들어갈 수 있지만 div 태그는 들어 갈 수 없다든지 하는 규칙을 정해주는 것이다. <br />
따라서, 브라우저는 이 DOCTYPE 을 보고 HTML문서를 어떤 방식으로 해석할지 결정하게 된다. <br />
만약 이 DOCTYPE 이 명기되어 있지 않으면 브라우저는 **quirks mode** 라는 모드로 HTML문서를 해석한다. 

## no-quirks mode vs quirks mode
앞서 브라우저의 HTML 해석모드에 quirks mode 라는 것이 있다고 언급했다. <br />
브라우저의 HTML 해석모드는 크게 3가지다. 

- quirks mode(호환 모드)
- no-quirks mode(standards 모드, 표준 모드)
- almost standards mode(거의 표준 모드)

이렇게 모드가 나뉘어 있는 이유는 옛날에 만들어진 아주 오래된 웹사이트에 대한 호환성을 지원하기 위함이다. <br />
예전의 HTML, CSS 스펙을 기준으로 하여 만들어진 오래된 웹사이트들은 현재의 표준을 지키지 못하기 때문에 모던 웹브라우저에서 렌더링이 잘못 되는 경우가 발생한다. <br />
이 때문에 HTML 문서를 오래전 스펙 기준으로 해석할 것인지 아닌지를 브라우저에게 알려주고 브라우저가 해당 모드로 해석하도록 만드는 것이다. <br />

## 선언 위치
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset=UTF-8>
    <title>Hello World!</title>
  </head>
  <body>
  </body>
</html>
```

HTML5 DOCTYPE 선언은 **`<!DOCTYPE html>`** 이다. HTML5 에서 권장하는 가장 간단한 형식의 선언이다. 더 복잡한 DOCTYPE 들도 있지만 특별한 경우가 아니라면 사용할 필요는 없다. <br />
해당 선언은 HTML 문서의 맨 첫 줄에 명시한다.
